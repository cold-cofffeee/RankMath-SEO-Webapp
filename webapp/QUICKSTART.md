## For XAMPP Users (Windows)

### Step 1: Installation
1. Your files are already in: `C:\xampp\htdocs\rankmath\webapp`
2. Start XAMPP Control Panel
3. Click "Start" for **Apache** and **MySQL**

### Step 2: Run Installer
1. Open browser
2. Go to: `http://localhost/rankmath/webapp/install.php`
3. Click "Check Requirements" button
4. If all green, click "Continue to Database Setup"
5. Enter database details:
   - **Host**: localhost
   - **Port**: 3306
   - **Database Name**: rankmath_webapp
   - **Username**: root
   - **Password**: (leave empty for XAMPP default)
   - **Prefix**: rm_
6. Click "Test Connection & Install Database"
7. Wait for installation to complete
8. Click "Go to Dashboard"

### Step 3: Start Using
You're now at: `http://localhost/rankmath/webapp/`

## Troubleshooting

### Apache won't start?
- Port 80 might be in use
- Close Skype or other apps using port 80
- Or change Apache port in XAMPP config

### MySQL won't start?
- Port 3306 might be in use
- Stop other MySQL instances
- Or change MySQL port in XAMPP config

### Can't see the website?
- Check Apache is running (green in XAMPP)
- Try: http://127.0.0.1/rankmath/webapp/
- Clear browser cache

### Database connection error?
- Make sure MySQL is running (green in XAMPP)
- Default XAMPP credentials: root / (no password)
- Database will be created automatically
